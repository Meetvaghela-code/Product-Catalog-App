Product Catalog System

A simple FastAPI + MongoDB product catalog demonstrating file uploads, CRUD APIs, and a lightweight frontend.

Pages: Add Product (Form) · Dashboard · Product Manager

What you'll learn

  • Designing RESTful APIs with FastAPI
  • Storing data in MongoDB (async with Motor)
  • Uploading and serving images
  • Building a responsive frontend with Bootstrap
How to run
  1. Install dependencies from requirements.txt
  2. Start MongoDB locally and run the FastAPI app with uvicorn
